Question 1

Take a number from a user and write a function which checks whether the number is even or odd. Assign the result to a variable and log that variable in the console.

Answer


                let getUserValue = prompt('Enter the value');
                getUserValue = Number(getUserValue);
        
                let varFunction = function (getUserValue) {
        
                    if (getUserValue % 2 != 0)
                        console.log(`The given number ${getUserValue} is Odd `)
                    else
                        console.log(`The given number ${getUserValue} is Even `)
        
                }
                varFunction(getUserValue);

Question 2

Write a program which will take OS name and version from the user separated by a space. Then log the OS name and version on the console.

Answer


                let getOSName = prompt('Enter the Os Name and its version with space');
